Alumni

Alumni dapat di tambahkan dengan cara satuan dan dapat pula menggunakan file csv.

Untuk penambahan manual, anda dapat mengakses menu Create -> Alumni.

Untuk penambahan secara CSV, anda dapat mengaksesnya pada dashboard -> Data Alumni di bagian bawah data alumni.

Untuk format menggunakan CSV, dapat dilihat seperti contoh dibawah

NOTE: Nama kolom dan value nya merupakan CASE SENSITIVE!

NIM Tipe Sekolah Gender Kota Sekolah Keterangan Lulus IPS_1 IPS_2 IPS_3 IPS_4
12.11.1111 SMK L Luar Kota Tidak Tepat Waktu 2.5 3.75 1.75 2.42

Untuk Tipe Sekolah terdapat tiga opsi, yaitu:

  • SMA
  • SMK
  • Lain -> Selain SMA dan SMK

Untuk Gender terdapat dua opsi, yaitu:

  • L
  • P

Untuk Kota Sekolah terdapat dua opsi, yaitu:

  • Luar Kota -> sekolah siswa bukan berasal dari Kota Yogyakarta
  • Dalam Kota -> sekolah siswa berasal dari Kota Yogyakarta

Note: Bukan DIY, namun hanya kota yogyakarta

Untuk Keterangan Lulus terdapat dua opsi, yaitu:

  • Tepat Waktu
  • Tidak Tepat Waktu

Untuk IP, silahkan masukkan bilangan bulat positif ataupun pecahan.


Training

Data training berasal dari semua data alumni yang digunakan untuk membuat model klasifikasi. Ketika model sudah dibuat, dan anda menambahkan data alumni, maka data alumni anda yang baru tidak akan dimasukkan ke dalam model.

NaN Error: Ketika model anda mengalami masalah NaN, solusinya sangat mudah, pastikan data dari excel anda memiliki kelas yang berbeda di setiap 10% dari total data anda.

Prediksi

Anda dapat memprediksi data baru menggunakan model yang sudah dibuat sebelumnya. Untuk melakukan prediksi data satu per satu, anda dapat mengaksesnya pada menu Prediksi -> Single Data.

Anda juga dapat memprediksi data menggunakan file csv apabila data yang akan diprediksi berjumlah banyak.

Untuk format menggunakan CSV, dapat dilihat seperti contoh dibawah

NIM Tipe Sekolah Gender Kota Sekolah IPS_1 IPS_2 IPS_3 IPS_4
12.11.1111 SMK L Luar Kota 2.5 3.75 1.75 2.42

Note: Untuk opsi di setiap kolomnya SAMA DENGAN milik alumni.
